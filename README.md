# Projeto do Curso Alura - Spring Boot 3: desenvolva uma API Rest em Java

## Objetivo
Nesse repositório foi criado uma API Rest com Spring Boot assim como no curso, porém, o curso tem um projeto sobre médicos e pacientes e nesse repositório foi feito um projeto diferente e mais simples.

## Técnicas aplicadas
- Criar uma API Rest em Java com Spring Boot;
- Realizar CRUD utilizando Banco de Dados MySQL;
- Flyway como ferramenta de Migrations da API;
- Validações de dados com Beam Validation;
- Paginação dos dados da API.

## Tecnologias
- Linguagem Java 17;
- Spring Boot
- Banco de dados MySQL;
- Docker: para simular a infraestrutura de banco de dados;
